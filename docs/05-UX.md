## Design Language

### Colors
- **Background**: Dark slate (#0a0f1a) with subtle gradients
- **Surface**: Elevated cards (#111827) with subtle borders (#1f2937)
- **Primary accent**: Electric blue (#3b82f6) — intelligence, clarity
- **Secondary accent**: Amber (#f59e0b) — highlights, alerts
- **Success/positive**: Emerald (#10b981)
- **Warning/negative**: Rose (#f43f5e)
- **Text primary**: White (#f9fafb)
- **Text secondary**: Slate (#94a3b8)
- **Text muted**: Dark gray (#64748b)

### Typography
- **Headings**: Inter (tight tracking, bold)
- **Body**: Inter (regular weight)
- **Data/numbers**: JetBrains Mono (monospace for stats)

### Component Patterns
- **Metric Badge**: Rounded pill with score + color coding (green/yellow/orange/red)
- **Confidence Indicator**: 3-5 dot scale or thin bar
- **Streak Tag**: Colored label with classification text
- **Stat Range**: "22.5 (19-26)" format — center with faded range
- **Trend Arrow**: Subtle up/down/flat indicator with delta value
- **Explanation Drawer**: Slide-out panel from right with structured explanation
- **Game Card**: Compact card with teams, score/projection, key tags
- **Metric Card**: Score + label + mini sparkline + confidence dots

## Responsive Strategy

| Breakpoint | Layout |
|-----------|--------|
| Desktop (1280px+) | Full dashboard with sidebar, 2-3 column grids |
| Laptop (1024px) | Sidebar collapses to icons, 2-column grids |
| Tablet (768px) | No sidebar, hamburger nav, single column, cards stack |
| Mobile (< 768px) | Full-width cards, simplified metric display, tab navigation |

Priority: Desktop > Laptop > Tablet > Mobile. Mobile is functional but not the primary experience.

## Reusable Component System

| Component | Usage |
|-----------|-------|
| `MetricBadge` | Everywhere metrics appear — score + color + label |
| `ConfidenceIndicator` | Next to every metric — dot scale or bar |
| `StreakTag` | Player cards, dashboards — colored streak label |
| `StatRange` | Projections — center value with range |
| `TrendArrow` | Leaderboards, metric cards — delta indicator |
| `GameCard` | Dashboard, games hub — compact game summary |
| `MetricCard` | Detail pages — score + sparkline + confidence |
| `ExplanationDrawer` | Detail pages — slide-out explanation panel |
| `DataTable` | Leaderboards, rosters — sortable, filterable |
| `RadarChart` | Compare, player/team detail — multi-metric overlay |
| `RollingChart` | Trend views — multi-window line chart |
| `SearchBar` | Global — player/team search with autocomplete |
| `WinProbBar` | Game cards — visual probability bar |
